Getting Started

Welcome to RO-EXEC! We 're thrilled to have you here. Let 's get started by installing the necessary files to run the loader. If you 're already set up with these files, skip this step.

Please make sure your Windows account is an ADMINISTRATOR, prior to continuing with the setup. If it is not, force it to be an Administrator or create a new local account.

Please install the following in order to ensure functionality of the software.

  • https://aka.ms/vs/17/release/vc_redist.x64.exe

  • https://aka.ms/vs/17/release/vc_redist.x86.exe

Check if your Windows is 64-bit version.

Open the search bar on your Taskbar, and search for "System Information ".

The system type should display "x64-Based PC "as shown in the image.

If there 's a problem with this method; Press Windows + I on your keyboard to open Settings, and then click on "System ".

Afterwards, scroll down and click on "About ".

Make sure it says "64-bit operating system "below the Device specifications.
